# **App Name**: RuleSage

## Core Features:

- Variable Extraction: Variable Extraction Tool: Use natural language processing to automatically identify key variables and their possible values from the user's descriptive text.
- Decision Tree Logic: Decision Tree Generation Tool: Construct a decision tree from extracted variables and values to represent the logical flow of the problem-solving process.
- Readable Explanation: Natural Language Output: Generate a human-readable version of the decision tree for non-technical users, explaining the decision-making process.
- Structured Data Export: JSON Output: Provide a structured JSON representation of the decision tree for the rules engine to interpret and execute.
- Smart Guidance: AI-Enhanced User Guidance: Guide users through the decision tree by asking sequential questions. Use AI to rephrase unclear answers or suggest related options to ensure accurate traversal.
- Professional Interface: Minimalist UI: Provide a clean, professional user interface focused on clarity and ease of use, suitable for step-by-step problem solving.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey trustworthiness and clarity.
- Background color: Light grey (#FAFAFA) for a clean, professional look.
- Accent color: Soft green (#81C784) to indicate positive outcomes and decisions.
- Body and headline font: 'Inter', a sans-serif font known for its clean readability, for both headlines and body text.
- Clean and structured layout for displaying variables, decision trees, and questions, ensuring easy navigation.
